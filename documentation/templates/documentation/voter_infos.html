Comment voter?

Le ticket de vote:

   Pour voter à une élection (dans un bureau de vote) vous devez disposer d'un ticket de vote.
Ce ticket, généré automatiquement lors de la création du bureau, vous est remis par l'organisateur de l'élection. Il permet d'identifier les électeurs de manière unique et spécifique tout en restant anonyme.

La procédure de vote:



Etape 1: Saisir le code du ticket

   Pour saisir le code de votre ticket entrez le directement dans le formulaire de la page d'accueil ou selectionnez l'onglet "Voter" dans la barre de menu.

Dans ce dernier cas un formulaire s'affichera à l'écran.

Tapez le code de votre ticket puis cliquez sur "Valider".

Etape 2: Et bien votez maintenant!

Si le code du ticket correspond bien à un bureau de vote et qu'il n'a jamais été utilisé, il sera détruit et un bulletin de vote s'affichera. (Et oui le ticket est détruit. :) Vous ne pensiez pas voter plusieurs fois tout de même?!)
Pour remplir un bulletin de vote de type "scrutin de Condorcet" vous devez ranger les candidats par ordre de préférence puis valider.

Si le bulletin est correctement rempli, un message de validatio s'affichera à l'écran.

Et voilà! Vous avez fini de voter. Il n'y a plus qu'à attendre les résultats.

Comment remplir un bulletin de Condorcet:

   C'est simple et flexible.
Vous devez ranger les candidats par ordre de préférences.
Les égalités de rangement sont autorisées et l'ordre des candidats n'est pas obligé d'être continu.

Exemple de bulletin valides:
Dans l'ordre: Candidat 1: 1, Candidat 2: 2, Candidat 3: 3, Candidat 4: 4, Candidat 5: 5

Dans le désordre: Candidat 1: 2, Candidat 2: 4, Candidat 3: 1, Candidat 4: 5, Candidat 5: 3

Avec des égalités: Candidat 1: 1, Candidat 2: 1, Candidat 3: 2, Candidat 4: 3, Candidat 5: 4

Avec des égalités et des trous dans les valeurs: Candidat 1: ,1 Candidat 2: 5, Candidat 3: 5, Candidat 4: 5, Candidat 5: 2


Rmq: Il est normalement possible dans un bulletin de Condorcet de ne pas classer un candidat. Chez Condorvote nous avons cependant fait le choix de contraindre les bulletins à contenir un rang pour chaque candidat. Nous pensons en effet qu'il est important pour un électeur de connaître (au moins un peu) chaque candidat afin de fournir un vote le plus éclairé possible.

Quelle est la meilleure stratégie de vote pour scutin de Condorcet:

   Aucune!
Enfin disons plutôt que la meilleure stratégie est d'indiquer vos vraies préférences. Peu importe le nombre de candidat d'un camp qui vous plaît, leurs alliances ou ce que les autres électeurs ont voté. Ce qui compte c'est l'ordre dans lequel vous ordonnez les candidats. Ainsi si vous préférez les candidats d'un camp par rapport à un autre, ils seront toujours tous classés au dessus des autres, et cela même s'il possède cent candidats et l'autre un seul.
Ce camps l'emportera donc toujours si suffisament d'électeurs pensent comme vous!
Pour plus de détail voir: les principes du scrutin de Condorcet

Les résultats:

   Pour avoir les résutats de votre bureau, vous devez attendre que l'organisateur de l'élection clôture le bureau et lance le calcul de l'élection.
Il vous transmettra alors lui même les résultats.
